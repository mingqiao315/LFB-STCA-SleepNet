Usage Method：
1. Download Dataset：link：https://pan.baidu.com/s/1CkVbK-JKcz0A2BpGs_BeTw
                     extraction code: jaky 
2. Run train.py directly. evaluation results will be output automatically after training completes.

If you want to perform evaluation from scratch starting with dataset processing, download the sleepedf files to the sleepedf folder and run dataset.py.

If it prompts that the dataset file is not found, modify the BATCH_METADATA_PATH parameter at the macro definition section in the train.py and pretrain.py codes to ensure the processed dataset path matches the configuration parameter.
